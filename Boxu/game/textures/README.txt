Splash intro — files in this folder are copied next to `gpframework` under `textures/` on build.

  autSplash1.(png|jpg|jpeg)    — first logo (hold, then fade out)
  boxuSplash4.(png|jpg|jpeg)   — second logo (fade in, hold, fade out)

If either file is missing, the intro is skipped and a log line explains why.

Other gameplay textures (e.g. board8x8.png) live here too.
